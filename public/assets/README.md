# Assets do projeto

Coloque aqui imagens, texturas e outros arquivos publicos do app.

Tudo dentro de `public` fica disponivel pela raiz do site. Exemplo:

```txt
public/assets/mana/blue.webp -> /assets/mana/blue.webp
```

Os botoes de mana procuram estes arquivos:

- `public/assets/mana/White.webp`
- `public/assets/mana/Water.webp`
- `public/assets/mana/Black.webp`
- `public/assets/mana/Red.webp`
- `public/assets/mana/Green.webp`
- `public/assets/mana/Colorless.webp`

Se alguma imagem nao existir, o app mostra a letra como fallback.
